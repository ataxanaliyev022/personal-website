About This Project

This is the GitHub repository for my personal website. The site is built with plain HTML, CSS, and JavaScript, with Jekyll support for static site generation. It features a clean, modern design focused on accessibility and maintainability.

Features

  • Responsive design that works on all devices
  • Smooth scroll animations and reveal effects
  • Accessible navigation and form handling
  • Clean, semantic HTML structure
  • Modern CSS with custom properties
  • Mobile-friendly navigation menu

Technologies

  • HTML5
  • CSS3 (with CSS Custom Properties)
  • Vanilla JavaScript
  • Jekyll (for static site generation)
